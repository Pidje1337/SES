# Односвязный список

| Операция    | Лушчий случай | Средний случай | Худший случай |
|:------------|:--------------|:---------------|:--------------|
| size()      | O(1)          | O(1)           | O(1)          |
| is_empty()  | O(1)          | O(1)           | O(1)          |
| clear()     | O(1)          | O(1)           | O(1)          |
| add_first() | O(1)          | O(1)           | O(1)          |
| add_last()  | O(1)          | O(N)           | O(N)          |
| insert()    | O(1)          | O(N)           | O(N)          |
| remove()    | O(1)          | O(N)           | O(N)          |
| pop()       | O(1)          | O(N)           | O(N)          |
| find()      | O(1)          | O(N)           | O(N)          |
