# Ассоциативный массив

| Операция        | Лучший случай | Средний случай | Худший случай |
|:----------------|:--------------|:---------------|:--------------|
| get_size()      | O(1)          | O(1)           | O(1)          |
| clear()         | O(1)          | O(1)           | O(1)          |
| is_empty()      | O(1)          | O(1)           | O(1)          |
| __hash()        | O(1)          | O(1)           | O(1)          |
| add()           | O(1)          | O(1)           | O(1)          |
| resize()        | O(1)          | O(1)           | O(1)          |
| get_by_key()    | O(1)          | O(1)           | O(1)          |
| insert_on_key() | O(1)          | O(1)           | O(1)          |
| delete_by_key() | O(1)          | O(1)           | O(1)          |
| print_keys()    | O(1)          | O(1)           | O(1)          |
| print_values()  | O(1)          | O(N)           | O(N)          |
| print_pairs()   | O(1)          | O(N)           | O(N)          |
