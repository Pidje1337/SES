# Очередь с приоритетом

| Операция   | Лучшмй случай | Средний случай | Худший случай |
|:-----------|:--------------|:---------------|:--------------|
| size()     | O(1)          | O(1)           | O(1)          |
| is_empty() | O(1)          | O(1)           | O(1)          |
| peek()     | O(1)          | O(1)           | O(1)          |
| enqueue()  | O(1)          | O(N)           | O(N)          |
| dequeue()  | O(1)          | O(1)           | O(1)          |
