# Двунаправленная очередь

| Операция       | Лучший случай | Средний случай | Худший случай |
|:---------------|:--------------|:---------------|:--------------|
| size()         | O(1)          | O(1)           | O(1)          |
| is_empty()     | O(1)          | O(1)           | O(1)          |
| clear()        | O(1)          | O(1)           | O(1)          |
| peek()         | O(1)          | O(1)           | O(1)          |
| peek_tail()    | O(1)          | O(1)           | O(1)          |
| e1queue()      | O(1)          | O(1)           | O(1)          |
| e1queue_head() | O(1)          | O(1)           | O(1)          |
| dequeue()      | O(1)          | O(1)           | O(1)          |
| dequeue_tail() | O(1)          | O(1)           | O(1)          |
